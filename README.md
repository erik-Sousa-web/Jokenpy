# Jokenpy# JokenPy

## Sobre o projeto

O JokenPy é um sistema web desenvolvido em Python utilizando o framework Flask. O objetivo do projeto é permitir que o usuário jogue Pedra, Papel e Tesoura contra o computador através de uma interface simples, moderna e intuitiva.

Este projeto foi desenvolvido como atividade prática da disciplina de Engenharia de Software, aplicando conceitos de metodologias ágeis, versionamento com Git, testes automatizados e integração contínua.

---

## Funcionalidades

- Escolher Pedra, Papel ou Tesoura;
- Jogar contra o computador;
- Exibir o resultado da partida;
- Interface responsiva.
